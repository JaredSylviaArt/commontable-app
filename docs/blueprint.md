# **App Name**: CommonTable

## Core Features:

- Secure Authentication: User authentication with email/password or Google login for church staff.
- Resource Listing: Create and browse listings for resources (Give, Sell, Share).
- Detailed Listings: Add title, description, category, location (ZIP), image, condition, and contact preference to listings.
- Direct Messaging: Enable threaded messaging between users for each listing.
- User Dashboard: Dashboard with user's active listings, messages, saved posts, and profile info.
- Moderation Tools: Admin moderation tools for flagging listings and adding admin-only notes.
- AI Category Suggestion: AI tool to suggest appropriate categories based on the listing title and description.

## Style Guidelines:

- Primary color: Deep teal (#2E8B8A) evoking trustworthiness and tranquility.
- Background color: Light cyan (#E0FFFF), very desaturated, to create a calm and inviting atmosphere.
- Accent color: Muted yellow-green (#9ACD32) for a natural and hopeful feel, drawing attention to calls to action.
- Headline font: 'Playfair', a modern sans-serif. Body font: 'PT Sans', a humanist sans-serif. Note: currently only Google Fonts are supported.
- Simple, hand-drawn style icons to represent different resource categories (gear, curriculum, creative assets).
- Clean, card-based layout for listings to prioritize a warm, ministry-minded user experience. Desktop-first, mobile responsive.
- Subtle transitions and animations for user interactions (e.g., message received, listing updated).
